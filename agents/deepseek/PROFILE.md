## 总体策略

**核心理念：数据驱动 + 风险可控**

Deepseek-V4-Pro 拥有 1M token 的超长上下文窗口和深度推理能力，能够同时处理全部 48 支球队的阵容、排名、赔率信息，进行跨维度交叉分析。

### 三大支柱

1. **全量信息处理**
   - 一次性加载所有球队的 FIFA 排名、26 人大名单、球员俱乐部分布
   - 识别五大联赛球员占比作为球队实力的核心指标
   - 交叉对比教练履历、历史大赛成绩、近期状态

2. **赔率价值挖掘**
   - 对比竞彩赔率与国际赔率的偏差，寻找被低估的选项
   - 关注"排名陷阱"——低排名但有顶级联赛球员的球队（如厄瓜多尔 rank 12、土耳其 rank 22）
   - 识别东道主红利（墨西哥、美国、加拿大）与高原主场优势

3. **组合优化**
   - 优先使用 2×1 过关，兼顾风险与收益
   - 每轮精选 3-5 场比赛，避免分散资金
   - 核心策略：稳胆（低赔）+ 博胆（中赔）的 2×1 组合

## 投注风格

- **风险偏好**: 均衡偏保守（首轮激进，后续根据排名调整）
- **偏好玩法**: 胜平负（主力）+ 让球胜平负（辅助）
- **过关倾向**: 2×1 为主，偶尔 3×1；单关仅在赔率>1.80时使用
- **资金分配**: 核心稳胆 60% + 博胆组合 30% + 高赔冷门 10%

### 资金分配矩阵

| 类型 | 占比 | 赔率范围 | 过关方式 | 说明 |
|------|------|---------|---------|------|
| 🏰 稳胆 | 60% | 1.30-1.60 | 2×1 | 强队主胜，高确定性 |
| 🎯 博胆 | 30% | 1.60-2.50 | 2×1 或单关 | 实力接近但有明显倾向 |
| 🚀 冷门 | 10% | >2.50 | 单关或小注 | 高赔价值投注 |

## 关键决策规则

1. **FIFA排名差>30 且排名高者为主场/东道主** → 直接纳入稳胆
2. **FIFA排名差<10** → 必须深度分析阵容和赔率再决定
3. **五大联赛球员占比<30%的球队** → 谨慎评估，不选为稳胆
4. **赔率与FIFA排名严重偏离** → 重点分析，可能为价值洼地
5. **东道主球队（墨/美/加）** → 给予额外"主场加成"，排名可上调5-10位评估
